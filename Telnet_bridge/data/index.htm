Telnet bridge configuration

Serial port

Configure the speed of the serial port. After saving the speed is directly changed.

Access point

IP:

Configure the SSID and password of the AccessPoint of the bridge. After saving the AccessPoint is directly changed.

Networks

IP:

Configure the SSID and password of the networks that the bridge connects to. There is a timeout of 20 seconds per network, then the next network is used. Networks with an empty SSID are not used. The network list is used only at startup of the bridge.
